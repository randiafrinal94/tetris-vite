# Tetris (Vite + React + Tailwind + Framer Motion)

## Quick Start
```bash
npm install
npm run dev
```

Open the printed local URL (usually http://localhost:5173).

## Controls
- Arrows: move
- ↑ / X: rotate clockwise
- Z: rotate counter‑clockwise
- Space: hard drop
- C: hold piece
- P: pause

Built with Vite, React 18, Tailwind CSS, and Framer Motion.
